# Hoops Sim Engine

A basketball simulation engine focused on realism, stats, and long-term league dynamics.

## MVP Goals

- Represent players, teams, and games as Python data models.
- Simulate a single game between two teams.
- Output a box score with key stats.

## Tech Stack

- Python 3.x

## Roadmap

### v0.1 - Core Engine
- Player model
- Team model
- Single game simulation
- Box score output

### v0.2 - Season Simulation
- Schedule generator
- Standings
- Basic playoffs

### v0.3 - Player Development
- Aging
- Potential

- Stats

### v0.4 - League Management
